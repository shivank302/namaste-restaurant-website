## 🎯 Project Overview

**Namaste Restaurant** is a beautifully crafted front-end project that simulates a real-world restaurant website experience.

It allows users to:

* Browse delicious Indian dishes 🍛
* Add items to cart 🛒
* Save favourites ❤️
* Explore a clean and modern UI

---

## ✨ Features

🌟 **User Experience Focused**

* 🍽️ Dynamic Menu Rendering
* ❤️ Add to Favourites System
* 🛒 Add to Cart Functionality
* 📄 Dish Details Page

📱 **Responsive Design**

* Works seamlessly on Mobile, Tablet & Desktop
